# Dental Clinic - Odoo 17 Module

## Description
Dental Clinic Management System for Odoo 17 with interactive dental chart.

## Features
- Patient registration with teeth selection
- Treatment management (Fixed, Removable, Simple)
- Payment tracking with remaining amount calculation
- Interactive SVG dental chart (Universal Numbering 1-32)
- Purchase management
- Profit reporting

## Installation
1. Copy module to `custom-addons/` folder
2. Update app list in Odoo
3. Install "Dental Clinic" module
